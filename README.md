## 🚀 Personal Projects

### 🤖 [MyBotMaker](https://mybotmaker.com/)

> **A no-code platform to build and deploy Telegram bots — no programming required.**

**MyBotMaker** is a personal SaaS product I designed and developed from scratch. It allows anyone to create fully functional Telegram bots through an intuitive visual interface, without writing a single line of code.

**✨ Key Features:**
- 🛠️ **Visual Bot Builder** — Drag-and-drop flow editor to design conversation flows
- 💬 **Multi-step Conversations** — Support for complex dialog trees and branching logic
- 📦 **Plugin System** — Built-in modules for common use cases (menus, forms, auto-replies, etc.)
- 🔗 **Webhook & API Integration** — Connect your bot to external services and APIs
- 📊 **Analytics Dashboard** — Track user interactions and bot performance in real time
- ☁️ **Managed Hosting** — Deploy your bot instantly without managing any infrastructure

**🔧 Tech Stack:** Node.js · MongoDB · Redis · Docker · Telegram Bot API

**🌐 Live:** [https://mybotmaker.com](https://mybotmaker.com/)
